```md
# Rock Paper Scissors Game
```

## Overview
A simple GUI-based Rock Paper Scissors game where:

- User plays against computer
- Scores are tracked
- Win streaks are displayed
- Reset option available

## Features
- Instant gameplay
- Score tracking
- Win streak counter
- Interactive GUI

## Technologies Used
- Python
- Tkinter
- Random module

## Installation
```bash
git clone https://github.com/your-username/rps_game.py
cd rps_game.py
```

## How to Run
```bash
python rps_game.py
```

## Gameplay
1. Click Rock, Paper, or Scissors
2. Computer selects random move
3. Result is displayed
4. Score updates automatically
