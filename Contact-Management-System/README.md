```md
# Smart Contact Manager
```

## Overview
A professional contact management system that allows users to:

- Add contacts
- Update contacts
- Delete contacts
- Search contacts
- Store data locally

## Features
- Add, update, delete contacts
- Search functionality
- Unique ID-based storage
- GUI interface

## Technologies Used
- Python
- Tkinter
- UUID module

## Installation
```bash
git clone https://github.com/your-username/contact_manager.py
cd contact_manager.py
```

## How to RUn
```bash
python contact_manager.py
```

## Usage
1. Add contact details
2. Search contacts
3. Select and update
4. Delete when needed
