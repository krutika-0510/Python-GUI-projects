# Smart Productivity Manager

## Overview
Smart Productivity Manager is a task management application built using Python and Tkinter. It helps users manage tasks efficiently by:

- Adding and tracking tasks
- Categorizing tasks
- Setting priorities
- Tracking due dates
- Marking tasks as completed
- Calculating productivity score
- Storing data persistently using JSON

## Features
- Task categories (Study, Work, Personal)
- Priority levels (High, Medium, Low)
- Due date tracking
- Overdue task detection
- Search functionality
- Productivity score
- JSON-based local storage

## Technologies Used
- Python
- Tkinter (GUI)
- JSON (Storage)
- Datetime module
