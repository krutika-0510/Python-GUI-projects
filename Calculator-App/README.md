```md
# Basic Calculator GUI
```

## Overview
This is a Tkinter-based calculator that supports:

- Basic arithmetic operations
- Calculation history
- Dark/light theme
- Keyboard input support

## Features
- Addition, subtraction, multiplication, division
- Calculation history
- Theme switching
- Backspace and clear
- Keyboard support
- Error handling

## Technologies Used
- Python
- Tkinter

## Installation
```bash
git clone https://github.com/your-username/calculator.py
cd calculator.py
```

## How to Run
```bash
python calculator_gui.py
```

## Usage
1. Click buttons or type expressions
2. Press = to calculate
3. View history
4. Toggle theme
